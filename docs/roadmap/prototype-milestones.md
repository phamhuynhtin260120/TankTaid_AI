# Prototype Milestones

## Milestone 1 - Core Combat

- Tank movement
- Turret aiming
- Projectile and damage
- Enemy bot behavior
- Mission win or lose

## Milestone 2 - Home Defense

- Grid-based home placement
- Walls and basic defensive turret
- Save and load home layout

## Milestone 3 - Raid Flow

- Load another home layout
- Raid objective
- Raid timer and result screen

## Milestone 4 - Backend Sync

- WebSocket room/session flow
- Player profile and home payload
- Raid result submission
- Reward and lucky spin result handling
