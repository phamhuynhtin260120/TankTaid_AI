# TankTaid Prototype Core Loop

1. Player enters `Garage` and equips a tank loadout.
2. Player runs a `Mission` against bots or bosses to earn currency and upgrade items.
3. Rewards improve tank stats or unlock `Home` defenses.
4. Player edits `HomeDefense` layout with walls, turrets, and defensive pieces.
5. Player starts a `Raid` on another player's home layout.
6. If the raid objective is destroyed before the player tank fails, the raid is a win.
7. Lucky spin rewards add a lightweight surprise layer during or after raids.

Prototype priority:

- First prove `Mission -> Reward -> Home Upgrade -> Raid`.
- Keep social and economy rules thin until the combat loop feels good.
